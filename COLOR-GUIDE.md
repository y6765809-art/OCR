# 🎨 מדריך צבעים - מערכת OCR תורנית

## ערכת הצבעים החדשה (ללא כחול!)

### 🟤 חום - הצבע העיקרי
```css
--brown-dark: #3E2723      /* Header, טקסט חשוב */
--brown-medium: #5D4037    /* גבולות, הדגשות */
--brown-light: #8D6E63     /* רקעים משניים */
```
**שימוש:** Header, ניווט, טקסט ראשי

### 🟡 זהב - יוקרה ומעמד
```css
--gold: #D4AF37            /* כפתורים ראשיים */
--gold-light: #F4E4C1      /* רקע כפתורים */
--gold-dark: #B8941F       /* Hover */
```
**שימוש:** כפתור "שמור", "המשך", סטטוסים חיוביים

### 🥉 ברונזה - חום ואצילי
```css
--bronze: #CD7F32          /* כפתורים משניים */
--bronze-light: #E8B88A    /* רקעים */
--bronze-dark: #A0522D     /* גבולות */
```
**שימוש:** כפתורים משניים, אייקונים

### 🔴 בורדו - חשוב ודחוף
```css
--burgundy: #800020        /* מחיקה, אזהרות */
--burgundy-light: #A0293A  /* Hover */
```
**שימוש:** כפתור "מחק", "בטל", אזהרות

### 🟢 זית - רגיעה והצלחה
```css
--olive: #6B8E23          /* הצלחה, השלמה */
--olive-light: #9ACD32    /* סטטוס חיובי */
```
**שימוש:** "הושלם", אישורים, התקדמות

### 🤍 נייטרליים חמים
```css
--cream: #FAF7F2          /* רקע ראשי */
--ivory: #FFFFF0          /* רקע משני */
--beige: #F5F5DC          /* פאנלים */
--sand: #E8DCC4           /* גבולות עדינים */
```

## דוגמאות שימוש

### Header
```css
background: var(--gradient-brown);
color: var(--gold);
```

### כפתור ראשי
```css
background: var(--gradient-gold);
color: var(--brown-dark);
```

### כפתור שמירה
```css
background: var(--olive);
color: white;
```

### כפתור מחיקה
```css
background: var(--burgundy);
color: white;
```

### כפתור משני
```css
background: var(--gradient-bronze);
color: white;
```

## השוואה: לפני ואחרי

| אלמנט | לפני (כחול) | אחרי (חום) |
|-------|------------|-----------|
| Header | כחול מלכותי | חום תורני |
| כפתור ראשי | זהב | זהב (נשאר) |
| כפתור משני | כחול בהיר | ברונזה |
| טקסט | נייבי כהה | חום כהה |
| הצלחה | ירוק | ירוק זית |
| שגיאה | אדום | בורדו |

## טיפים לשימוש

1. **Header תמיד חום**: `var(--gradient-brown)`
2. **כפתור חשוב ביותר**: `var(--gradient-gold)`
3. **כפתור רגיל**: `var(--gradient-bronze)`
4. **מחיקה/ביטול**: `var(--burgundy)`
5. **הצלחה**: `var(--olive)`
6. **רקע**: `var(--cream)` או `var(--ivory)`

זה נראה **תורני, יוקרתי, ולא קלוד!** ✨
